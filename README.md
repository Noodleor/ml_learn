# ml_learn

В этом задании требуется проанализировать данные, выделить из них полезные признаки и обучить линейную модель. Среди используемых библиотек:  `sklearn`, `pandas`, `numpy`, `matplotlib`, `seaborn`, `folium`. Датасет "New York City Taxi Trip Duration" взят из https://www.kaggle.com/c/nyc-taxi-trip-duration/overview.
